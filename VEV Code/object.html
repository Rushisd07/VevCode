Object

What is object ?
     -It is a physical reality.
     -Object is an instance of a class.   
     -It is Consume Memory.
     -Every object is Unique identifier by hashcode.
     For Ex.car,bike,chair,pen. 
 
Syntax Of Object:
     ClassName ObjectName= new ClassName();

     ClassName:  Class that we have created for creating an object.
     ObjectName: Name of the object that we want to create.
     new:        The new is keyword that is use to assign the memory for the object.
     ClassName():It is a Constructor.

Characteristic of Object:
     It is three characteristic of object.
     Identity:It gives a unique name to an object.for Ex.Dog
     state:   It is represents by attributes of an object.for Ex.breed,Age,Color
     Behavior:It is represented by the methods of an object.for Ex.Bark,sleep,eat.
