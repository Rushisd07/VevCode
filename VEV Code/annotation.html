Spring Annotations

Annotations & Description & Their Packages
 
Annotation :
    - They are alternative to XML.
    - They are java classes either defined in java or frameworks.
    - Annotation should always start with @ followed by uppercase letter.
    - Following are the spring core annotations.
        
    There 2 types of annotations:
        1. Class level annotations
        2. Method level annotations

    

        

            Annotation
            Description
            Package
        

        

            @Controller 
            It is used to annotate a Spring MVC controller class.
            org.springframework.stereotype.Controller;
        

        

            @Service 
            It indicate that a class belongs to service layer
            org.springframework.stereotype.Service;
        

        

            @Repository 
            It is responsible for accessing and manipulating data 
from a database
            org.springframework.stereotype.Repository;
        

        

            @Component 
            It is a Father of all annotations. Used to create a bean.
            org.springframework.stereotype.Component;
        

        

            @RestController 
            Used to create restfull web service. Combination of 
@Controller + @ResponseBody
            org.springframework.web.bind.annotation.RestController;
        

        

            @GetMapping 
            Retrieve data from the server. It maps to the specific
handler method
            org.springframework.web.bind.annotation.GetMapping;
        

        

            @PostMapping 
            It send data to the server. Post request method to specific 
handler method
            org.springframework.web.bind.annotation.PostMapping;
        

        

            @RequestMapping 
            Used to map the web request to handler method.
            org.springframework.web.bind.annotation.RequestMapping;
        

        

            @PathVariable 
            Retrieve data from the URL path
            org.springframework.web.bind.annotation.PathVariable;
        

        

            @RequestParam 
            Used to extract the data from the query parameter.
            org.springframework.web.bind.annotation.RequestParam;
        

        

            @Transactional 
            Used to manage database transactions in spring Appln. 
It used to prevent repetitive code.
            
            org.springframework.transaction.annotation.Transactional;
        

        

            @Autowired 
            Used to create bean and it inject values automatically.
            org.springframework.beans.factory.annotation.Autowired;
        

        

            @ResponseBody 
            Instead of rendenring to view page it directly write in 
the response body. It is usefull for returning JSON, XML
or other types of Data.
            org.springframework.web.bind.annotation.ResponseBody;
        

        

            @ModelAttribute 
            It inject form data into the bean.
            org.springframework.web.bind.annotation.ModelAttribute;
        

        

            @Table 
            Used to specify the name of the database table. 
That the entity class is mapped to.
            javax.persistence.Table;
        

        

            @Column 
            Used to map a field in a java class to a column in DB table 
            javax.persistence.Column;
        

        

            @Id 
            Used to mark a field in a java class as a primary key in 
DB table.
            javax.persistence.Id;
        

        

            @Entity 
            Used to map a java class with DB table
            javax.persistence.Entity;
        

        

            @GeneratedValue 
            it is used to generate column id automatically
            javax.persistence.GeneratedValue;
