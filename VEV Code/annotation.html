Spring Annotations

Annotations & Their Packages
 
Annotation :
    - They are alternative to XML.
    - They are java classes either defined in java or frameworks.
    - Annotation should always start with @ followed by uppercase letter.
    - Following are the spring core annotations.
        
    There 2 types of annotations:
        1. Class level annotations
        2. Method level annotations
