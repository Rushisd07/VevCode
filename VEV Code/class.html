Class and Constructor

Class:
     -It is Organise relevent things.
     -It is user define datatype.
     -It is a blueprint or template for creating object.
     -It Contains properties and methods.
      For Ex.with bank of multiple class.
Syntax Of Class:
      access modifier class ClassName{
           //properties       -to hold data
           //constructor      -default overloaded
           //method(function) -To process data
      }
          

Construtor:
      -It is special function.
      -Identical name as that of the class name.
      -It is used to initialize/build/construct the properties.
      -To their default values/assigned values.
      -used in allocation in memory.
Type of constructor:
      1.Default constructor
      2.Parameter Constructor(overloaded constructor)
Default constructor:
      -A constructor that has no parameters is known as default constructor.
      -default constructor provides the dafault values to the object like null,0 etc.depending on the type.
       for Ex. Student(){                                     //Student is a class name
                 super();                                     //to invoke the super class method
                 system.out.println("Deafault constructor");  //it is used to print message on console
              }
parameter constructor:
      -A constructor that has parameters is known as parameterized constructor.
      -If we want to initialize field of the class with our own values,then use a parameterized constructor.

       for Ex.
            Student(int stdId,String stdName,...){  //Student is a class and stdId,stdName is a properties of class.
                    super();                          //to invoke the super class method
                    this.stdId=stdId;                 //this keyword is use to current object in method or constructor. 
                    this.stdName=stdName;
                     :            :
               system.out.println("parameteries constructor");
              }
