Spring Boot 3.x

Spring Boot

Features of Spring Boot:
    - Intializer.           --- Setting up of the Infrastructure ( start.spring.io )
            (SNAPSHOT : Beta Version, M1: Milestone (Can Also be used), RC: Release Version)
    - Starter Projects.     --- Means All the dependencies in one file  
    - Auto Configuration    --- Configures DispatcherServlet / VR / etc...
    - Profiles (Special Concepts in spring-boot only)
    - Actuator.

Disadvantages in Spring Framework:
    - Time-consuming Configurations/Dependencies.
    - Less Felxibility.

    @SpringBootApplication : (Interview Questions)
        - It's similar to declaring a class with @Configuration, @EnableAutoConfiguration, and @ComponentScan annotations.

    Which package/library helps for auto connfiguration in spring boot?(Interview Question)
        - spring-boot-autoconfigure package

Logging Levels 

    trace   ---	Complete Information Displayed,Developer Environment
    debug   ---	Developer Environment
    info    --- Production Environment,Important information displayed
    error   --- Gives Information Related To Error
    off     --- Given

Profiles:
    - Profiles are an Integral part of Spring Boot Framework as it allows Production Ready Applications Quickly.
    - We can easily push the application from one environment to another by making the appropriate Profile Active.
    - Appplication Maintainance becomes easy with Profiles and Configuration.    

MVC
    - Properties file:
    - For passing the prefix and suffix in the VR we use:
            - spring.mvc.view.prefix=/WEB-INF/pages/
            - spring.mvc.view.suffix=.jsp

Note:
        - JSP ( View Layer ) can be created in the following folder structure
            Under src/main/resources
        - Create the following folder structure
            META-INF/resources/WEB-INF      ---Spring Boot takes this PAth for JSP Files
        - jsp files can be placed in the above folder
